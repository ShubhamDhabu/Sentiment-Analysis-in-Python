# Sentiment Analysis in Python

This project performs Sentiment Analysis on text data using Python, NLP preprocessing, and machine learning models.  
It classifies text into **Positive**, **Negative**, or **Neutral** sentiment.

## Features
- Text cleaning and preprocessing  
- TF-IDF or Bag-of-Words feature extraction  
- Machine learning classification (Naive Bayes / Logistic Regression / SVM)  
- Model training, evaluation, and prediction  
- Script to test custom text sentiment  

## Project Structure
- preprocess.py — text cleaning functions  
- train.py — trains the sentiment model  
- sentiment.py — predicts sentiment for input text  
- data/ — dataset folder  
- models/ — saved ML model  
- requirements.txt — dependencies


## Example Output
Input:  
`This product is terrible!`  

Output:  
`Sentiment: Negative`

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- NLTK / SpaCy
